## 📌 Project Overview

A simulated IT helpdesk environment modelling real-world support operations for a company of ~50 employees.
Built to demonstrate systematic IT support thinking, documentation skills, and data-driven performance tracking.

## 🎯 Objectives

- Design a realistic ticket triage and escalation workflow
- Create standard operating procedures (SOPs) for common IT issues
- Build a KPI dashboard tracking helpdesk performance metrics
- Develop an IT asset inventory management system

## 📋 Ticket Categories Covered

| Category | Example Issues | Priority Level |
|---|---|---|
| Hardware | Laptop won't boot, monitor issues, keyboard/mouse | P2-P3 |
| Software | Application crashes, installation errors, licence issues | P2-P3 |
| Network | No internet, VPN issues, slow connection | P1-P2 |
| Access | Password reset, account lockout, MFA issues | P1-P2 |
| Onboarding | New user setup, device config, software install | P2 |

## 📊 KPI Dashboard Metrics

- Total tickets opened / resolved / pending
- Average resolution time by category
- First-contact resolution (FCR) rate
- SLA compliance rate (% resolved within target time)
- Customer satisfaction (CSAT) score per ticket
- Recurring issue trend analysis
